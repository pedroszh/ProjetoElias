Sobre o Projeto

Este gerador de receitas aleatórias foi criado para ajudar os usuários a encontrar receitas de maneira simples e divertida. Cada vez que o botão "Gerar Nova Receita" é clicado, uma nova receita é exibida, juntamente com seus ingredientes.

Objetivos do Projeto

O objetivo principal deste projeto é fornecer uma experiência interativa, onde os usuários possam explorar diferentes receitas de maneira prática. Além disso, o código foi escrito para ser claro e de fácil manutenção.

O projeto utiliza HTML, CSS e JavaScript de forma integrada para criar uma página dinâmica.
